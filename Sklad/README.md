## После настройки — 3 кнопки

| Действие | VS Code |
|----------|---------|
| Телефон (скан) | **F5** |
| ПК (браузер) | **Ctrl+Shift+P** → **💻 Веб: сервер + браузер** |
| APK на телефон | **Ctrl+Shift+P** → **📦 Flutter: собрать APK** |

Первый раз: **Ctrl+Shift+P** → **⚙️ Один раз: установить Flutter**
