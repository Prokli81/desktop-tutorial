# Sklad — склад мебели и техники

Первая версия (MVP): скан на **Android-телефоне**, подтверждение **всей накладной** на **ПК**.

## Запуск

```bash
cd Sklad && python3 -m http.server 4174
```

- **Телефон:** `http://<IP-вашего-ПК>:4174` → режим «Телефон (склад)»
- **ПК:** `http://localhost:4174` → режим «ПК (офис)»

Для камеры на телефоне нужен **HTTPS** или `localhost`. В локальной сети по HTTP камера может не открыться — тогда вводите код вручную.

## Как пользоваться

### Приёмка

1. На телефоне: **Приёмка** → сканируйте этикетки (или введите код, например `KOM-204-DUB`).
2. На ПК: раздел **Накладные** → выберите накладную → **Подтвердить всю накладную**.
3. Остатки появятся в **Остатки**.

### Отгрузка

1. На телефоне: **Отгрузка** → сканируйте упаковки.
2. На ПК: укажите **номер заказа** и **клиента** → **Подтвердить всю накладную**.

### Справочник и этикетки

- **Справочник** — модели изделий (до сотен позиций, есть поиск).
- **Этикетки** — правила разбора кодов и словари типов/цветов (редактируются).

## Синхронизация

- **Локально:** данные в браузере; на одном ПК два окна (телефон + ПК) синхронизируются автоматически.
- **Облако (опционально):** скопируйте `firebase-config.example.js` → `firebase-config.js`, укажите **свой** проект Firebase для Sklad. Данные синхронизируются между телефоном и ПК через Firestore. MyFitClub — отдельное приложение, не связано со Sklad.
