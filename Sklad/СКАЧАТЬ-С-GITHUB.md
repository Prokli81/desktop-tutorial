# Как получить файлы с GitHub (для новичка)

Тебе нужна папка **`C:\Users\Grimm\sklad`** с актуальными файлами.  
Есть **2 способа** — выбери тот, который проще.

---

## Способ 1 — через VS Code (если проект уже открыт)

1. Открой **VS Code** (файл `sklad.code-workspace` или просто VS Code из Пуска).
2. Нажми **Ctrl+Shift+P** (сверху появится строка поиска).
3. Набери: **`задачи`**
4. Выбери: **Задачи: выполнить задачу**
5. Выбери: **🔄 Обновить с GitHub**

Снизу в терминале побегут строки. Когда закончится — файлы обновлены.

**Если пишет ошибку** (нет git, не репозиторий) → используй **Способ 2**.

---

## Способ 2 — скачать ZIP (самый простой, всегда работает)

### Шаг 1. Открой ссылку в браузере

https://github.com/Prokli81/desktop-tutorial/tree/cursor/sklad-warehouse-app-9401

### Шаг 2. Скачай архив

1. Нажми зелёную кнопку **Code** (Код).
2. Выбери **Download ZIP** (Скачать ZIP).
3. Сохранится файл `desktop-tutorial-cursor-sklad-warehouse-app-9401.zip` (имя может чуть отличаться).

### Шаг 3. Распакуй

1. Правый щелчок по ZIP → **Извлечь всё**.
2. Откроется папка вроде `desktop-tutorial-cursor-sklad-warehouse-app-9401`.
3. Внутри найди папку **`Sklad`**.

### Шаг 4. Скопируй в нужное место

**Вариант А — заменить всё (проще):**

1. Переименуй старую папку `C:\Users\Grimm\sklad` в `sklad_старый`.
2. Скопируй папку **`Sklad`** из архива в `C:\Users\Grimm\`.
3. Переименуй скопированную **`Sklad`** в **`sklad`** (маленькими буквами).

Должно получиться:
```
C:\Users\Grimm\sklad\sklad.code-workspace
C:\Users\Grimm\sklad\НАЧНИ-ЗДЕСЬ.md
C:\Users\Grimm\sklad\sklad_flutter\
```

### Шаг 5. Открой в VS Code

Двойной щелчок: **`C:\Users\Grimm\sklad\sklad.code-workspace`**

---

## Способ 3 — клонировать через VS Code (если папки sklad ещё нет)

1. VS Code → **Файл** → **Открыть папку** (или слева иконка проводника).
2. Или: **Ctrl+Shift+P** → **Git: Clone**
3. Вставь ссылку:
   ```
   https://github.com/Prokli81/desktop-tutorial.git
   ```
4. Выбери папку `C:\Users\Grimm\`
5. После клонирования открой папку **`Sklad`** внутри репозитория.
6. Переключи ветку: снизу слева в VS Code нажми на название ветки → выбери **`cursor/sklad-warehouse-app-9401`**
