# Быстрый старт

**Полная инструкция для новичка:** открой файл **`НАЧНИ-ЗДЕСЬ.md`** (он в этой же папке).

Три действия после настройки:

| Что | Как |
|-----|-----|
| Телефон | **F5** |
| ПК в браузере | **Ctrl+Shift+P** → **Задачи: выполнить задачу** → **💻 Веб: сервер + браузер** |
| APK файлом | **Ctrl+Shift+P** → **📦 Flutter: собрать APK** |

Первый раз: **⚙️ Один раз: установить Flutter**
