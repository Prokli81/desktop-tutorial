# Android Studio — первый раз (для тех, кто ничего не настраивал)

Я **не могу нажимать кнопки на вашем ПК**, но этот файл + скрипт **`nastroyka-android.bat`** делают почти всё сами.

---

## Что вы делаете (3 действия)

### Действие 1 — один раз открыть Android Studio

1. Пуск → **Android Studio** → запустить  
2. Если спросит **Import settings** → **Do not import settings** → OK  
3. Мастер настройки:
   - **Next**
   - **Standard** (стандартная установка)
   - **Next** → выберите тему (любую)
   - **Next** → **Finish**
4. Внизу пойдёт **Downloading components** — **подождите до 100%** (10–30 минут, интернет нужен)
5. Когда готово — можно **закрыть** Android Studio

Больше в Studio ничего настраивать не нужно для первого APK.

### Действие 2 — запустить автонастройку Sklad

Проводник → `C:\Users\Grimm\sklad` → двойной щелчок:

**`nastroyka-android.bat`**

Скрипт сам:
- найдёт Android SDK  
- создаст `local.properties`  
- выполнит `npm install`  
- подготовит проект  
- попробует собрать **app-debug.apk**  
- откроет Android Studio, если нужно  

Следуйте подсказкам в чёрном окне (Enter после шагов).

### Действие 3 — APK на телефон

Если скрипт напишет **APK ГОТОВО**:
- откроется папка с файлом **`app-debug.apk`**
- скопируйте на телефон → установите → разрешите камеру

---

## Если скрипт не собрал APK сам

Откроется Android Studio с проектом. Подождите **Gradle Sync Finished** (внизу).

Меню **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**.

Готово → **locate** → `app-debug.apk`.

---

## Нужно ли Node.js?

**Да.** Если скрипт пишет «Node.js не установлен»:
https://nodejs.org/ → LTS → Install → новый PowerShell → снова `nastroyka-android.bat`.

---

## Ошибки

| Ошибка | Решение |
|--------|---------|
| SDK не найден | Дождитесь загрузки в Android Studio (действие 1) |
| Gradle failed | Откройте Studio → Build APK вручную |
| Node не найден | Установите Node.js LTS |
| Нет папки android | Обновите Sklad из ZIP репозитория |
