# Sklad как приложение на Android

## Два уровня

| Уровень | Что это | Сложность |
|---------|---------|-----------|
| **1. PWA** | Иконка «Sklad» на главном экране, без магазина | Уже в проекте |
| **2. APK** | Файл `.apk` как обычное приложение | Позже (Capacitor) |

---

## Уровень 1 — установить сейчас (PWA)

PWA = тот же Sklad, но открывается **без полосы браузера**, как приложение.

### Условия

1. Открывать по **https://** (ссылка из `start-s-kameroy.bat` + cloudflared)
2. Chrome на Android

### Установка на телефон

**Способ А — кнопка в Sklad**

1. Откройте https-ссылку на телефоне  
2. Появится плашка **«Установить»** → нажмите  
3. Иконка **Sklad** на рабочем столе  

**Способ Б — вручную в Chrome**

1. Меню **⋮** → **Установить приложение** / **Добавить на главный экран**  
2. Подтвердите  

Дальше запускайте **Sklad с иконки**, не из закладок.

### Камера в PWA

В установленном PWA камера работает так же, как в Chrome по **https://** — туннель cloudflared на ПК должен быть запущен (или постоянный HTTPS-хостинг).

## Уровень 2 — настоящий APK (позже)

Когда понадобится файл `.apk` (раздача без ссылки, иконка в списке приложений без Chrome):

- обёртка **Capacitor** вокруг текущего Sklad;
- нативный доступ к камере;
- сборка в Android Studio или облачной сборке.

Это **этап 4** по ТЗ. Сначала закрепим PWA + HTTPS на складе.

---

## Схема для вас

```
ПК:  start-s-kameroy.bat  →  https://....trycloudflare.com
Телефон:  открыть https → Установить → иконка Sklad
Скан:  камера в приложении Sklad
Подтверждение:  ПК localhost:4174 → ПК (офис)
```
