# Почему камера не работает на телефоне

## Причина

Chrome на Android **не даёт камеру** по обычному адресу:

```
http://192.168.x.x:4174
```

Это правило безопасности браузера: камера только на **https://** или на `localhost`.

Sklad открыт по **HTTP** в домашней Wi‑Fi — поэтому камера блокируется. **Это не поломка Sklad.**

## Что делать сейчас (без настройки)

Поле **«Код вручную»** под сканером — введите этикетку, например `KOM-204-DUB`, нажмите **Добавить**. Логика та же, что при скане.

## Как включить камеру позже

Нужен доступ по **HTTPS**. Варианты:

1. **Туннель** (проще для теста): ngrok, Cloudflare Tunnel — даёт ссылку `https://...`
2. **Свой HTTPS** на ПК (сложнее): сертификат + сервер с SSL
3. **Публикация** Sklad на хостинге с HTTPS (Firebase Hosting и т.д.)

Пока для склада достаточно **ручного ввода** или **USB-сканера штрихкодов** на ПК (этап позже).
