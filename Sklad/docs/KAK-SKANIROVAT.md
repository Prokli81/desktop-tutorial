# Как сканировать в Sklad

## Сейчас (без настройки HTTPS)

### 1. Ввод с телефона вручную
Поле **«Код вручную»** → введите этикетку → **Добавить**.  
Подходит, если этикеток немного или код можно прочитать глазами.

### 2. USB-сканер штрихкодов на ПК (лучший вариант для склада)
Купить **USB-сканер** (~1500–3000 ₽). Подключается как клавиатура:
- открыть Sklad на ПК в режиме **Телефон (склад)** или поле ввода кода;
- навести сканер на штрихкод → код сам попадёт в поле → Enter.

Камера и HTTPS **не нужны**.

---

## Скан камерой телефона (нужен HTTPS)

Chrome **не даёт камеру** по `http://192.168.x.x`. Нужен адрес **`https://...`**.

### Вариант А — бесплатный туннель ngrok (для теста)

1. Зарегистрироваться: https://ngrok.com/
2. Скачать ngrok для Windows, положить в PATH или в `C:\Users\Grimm\sklad`
3. На ПК сервер уже запущен: `python -m http.server 4174`
4. В **другом** окне PowerShell:

```powershell
ngrok http 4174
```

5. Скопировать строку **Forwarding** вида `https://xxxx.ngrok-free.app`
6. Открыть **эту https-ссылку** на телефоне → **Телефон (склад)** → камера должна работать.

> Бесплатный ngrok меняет ссылку при каждом запуске. Для постоянной работы — хостинг с HTTPS.

### Вариант Б — выложить Sklad в интернет с HTTPS
Firebase Hosting, GitHub Pages, свой VPS — постоянный `https://ваш-домен` для телефона и ПК.

### Вариант В — позже в программе
Можно добавить в Sklad встроенную подсказку и кнопку «открыть через HTTPS» (этап 3 по ТЗ).

## Что выбрать

| Способ | Сложность | Для склада |
|--------|-----------|------------|
| Ручной ввод | Нулевая | Мало этикеток |
| USB-сканер на ПК | Купить сканер | **Очень удобно** |
| ngrok + телефон | 15 мин настройки | Тест камеры на телефоне |
| HTTPS-хостинг | Средняя | Постоянная работа с телефона |

---

## Рекомендация

1. **Сейчас:** ручной ввод или USB-сканер.  
2. **Если нужна именно камера телефона:** ngrok на 15 минут для проверки.  
3. **На будущее:** выложить Sklad на HTTPS-хостинг.
