# Скачать приложение на телефон (без Flutter)

APK собирается автоматически на GitHub. Flutter на ПК **не нужен**.

## Шаг 1. Открой страницу сборок

https://github.com/Prokli81/desktop-tutorial/actions/workflows/sklad-apk.yml

## Шаг 2. Найди зелёную галочку

Сверху список сборок. Нужна строка со статусом **зелёная галочка** (успех).

Если все красные — напиши в чат, пересоберём.

## Шаг 3. Скачай APK

1. Нажми на **зелёную** сборку.
2. Прокрути вниз до блока **Artifacts**.
3. Скачай **sklad-app-debug** (это ZIP-файл).

## Шаг 4. Установи на телефон

1. Распакуй ZIP.
2. Внутри файл **`app-debug.apk`**.
3. Скопируй на телефон (USB, Telegram, почта).
4. На телефоне открой файл → **Установить**.
5. Если спросит «Разрешить установку из неизвестных источников» — разреши.

---

## Обновить проект в VS Code (ты подключил GitHub)

1. Открой `sklad.code-workspace` в VS Code.
2. Слева иконка **ветки** (Система управления версиями) или **Ctrl+Shift+G**.
3. Нажми **⋯** (три точки) → **Получить** (Pull).
4. Или в терминале снизу: `git pull`

После pull появятся новые файлы и инструкции.
