# Sklad — полная инструкция для новичка (VS Code на русском)

Ты ничего не должен знать заранее. Делай **по порядку**, не перескакивай шаги.

---

## ШАГ 0. Открыть проект

1. Открой папку `C:\Users\Grimm\sklad` в проводнике Windows.
2. Найди файл **`sklad.code-workspace`**.
3. **Двойной щелчок** по нему.

**Если открылся Блокнот**, а не VS Code:
- Правый щелчок по `sklad.code-workspace` → **Открыть с помощью** → **Visual Studio Code**.

Должно открыться окно **Visual Studio Code** (синяя иконка).

| Место | Как найти |
|-------|-----------|
| **Файлы слева** | Иконка **двух листов** на левой панели (Проводник) |

Слева должны быть **две папки**:
- **Sklad — Flutter** (приложение на телефон)

## ШАГ 1. Установить расширения (один раз)

### Способ А — если всплыло уведомление

Справа внизу может появиться: *«Это рабочая область рекомендует расширения»*  
→ нажми **Установить все** (или **Install All**).

### Способ Б — вручную

1. Слева нажми иконку **кубиков** (Расширения).
2. Вверху может быть блок **Рекомендуемые** → **Установить рабочую область** / **Установить все**.

Нужны: **Flutter**, **Dart**, **Russian Language Pack** (русский язык).

3. **Закрой VS Code полностью** (крестик) и **открой снова** `sklad.code-workspace`.
4. Если появится **«Перезагрузить окно»** — нажми **один раз**. После обновления с GitHub это больше не должно повторяться.

> **Не открывай `sklad.code-workspace` дважды** — иначе будет «рабочая область уже открыта». Если VS Code уже работает — просто продолжай в нём.

## ШАГ 2. Приложение на телефон — выбери один способ

### Способ А — БЕЗ установки Flutter (проще)

**Инструкция:** открой файл **`СКАЧАТЬ-APK.md`** в этой папке.

Кратко:
1. https://github.com/Prokli81/desktop-tutorial/actions/workflows/sklad-apk.yml
2. Зелёная галочка → **Artifacts** → **sklad-app-debug**
3. Распакуй → `app-debug.apk` → на телефон → установить

**Перейди сразу к ШАГ 4** (веб на ПК). Flutter не нужен.

### Способ Б — установить Flutter (для F5 и разработки)

**Самый простой:** двойной щелчок по файлу **`УСТАНОВИТЬ-FLUTTER.bat`** в папке `sklad`. Жди до конца (~1 ГБ).

**Или через VS Code:**

1. **Ctrl+Shift+P** → **Задачи: выполнить задачу** → **⚙️ Один раз: установить Flutter**
2. Жди в терминале до конца.
3. **Закрой VS Code** и **открой заново**.

### Проверка

1. **Ctrl+Shift+P**
2. Набери: **`flutter doctor`**
3. Выбери: **Flutter: Run Flutter Doctor**
4. Смотри терминал снизу:
   - **Жёлтый** — нормально, можно продолжать.
   - **Красный** — сделай скриншот терминала и пришли в чат.

## ШАГ 3. Запустить на телефоне (скан этикеток)

### Подготовка телефона

1. Подключи Android к ПК **USB-кабелем**.
2. На телефоне: **Настройки** → **Для разработчиков** → включи **Отладка по USB**.
   - Если нет «Для разработчиков»: **Настройки** → **О телефоне** → 7 раз нажми **Номер сборки**.
3. Когда телефон спросит «Разрешить отладку?» → **Разрешить** (можно поставить галочку «Всегда»).

### Запуск в VS Code

**Самый простой способ — клавиша F5:**

1. Нажми **F5** на клавиатуре.
2. Если спросит конфигурацию — выбери **📱 Sklad на телефоне**.
3. Снизу в терминале пойдёт сборка (первый раз 5–15 минут).
4. На телефоне откроется приложение **Sklad**.

**Если F5 не сработал:**

1. **Ctrl+Shift+P** → **Задачи: выполнить задачу**
2. Выбери **📱 Flutter: запустить на телефоне**

## ШАГ 4. ПК — подтвердить накладные в браузере

1. **Ctrl+Shift+P** → **Задачи: выполнить задачу**
2. Выбери **💻 Веб: сервер + браузер**
3. Откроется браузер с адресом `http://localhost:4174`
4. На странице выбери **ПК (офис)** → вкладка **Накладные** → **Подтвердить**

Чтобы остановить сервер: в терминале снизу нажми **Ctrl+C**.

---

## ШАГ 5. APK без USB (установить файлом на телефон)

1. **Ctrl+Shift+P** → **Задачи: выполнить задачу**
2. **📦 Flutter: собрать APK**
3. Готовый файл:
   ```
   C:\Users\Grimm\sklad\sklad_flutter\build\app\outputs\flutter-apk\app-debug.apk
   ```
4. Скопируй на телефон (USB, Telegram, почта) → открой → **Установить**.

## Частые проблемы

| Что видишь | Что делать |
|------------|------------|
| `flutter не является командой` | Повтори **ШАГ 2**, перезапусти VS Code |
| `No devices found` | USB-кабель, отладка по USB, «Разрешить» на телефоне |
| `python не найден` | Установи Python с python.org (галочка **Add to PATH**) |
| Терминал красный текст | Скриншот **всего** нижнего окна → в чат |
