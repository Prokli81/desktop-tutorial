# MyFitClub: фото прогресса в дневнике (бесплатно)

Фото хранятся в **Firebase Storage** — бесплатный лимит Spark хватает для клуба.

---

## Шаг 1. Включить Storage

1. https://console.firebase.google.com/project/myfitclub-4d1b8/storage
2. **«Начать»** / Get started
3. Режим: **«Начать в тестовом режиме»** (для первого раза)
4. Регион — тот же, что у Firestore (Europe)
5. **«Готово»**

---

## Шаг 2. Правила Storage (безопасность)

1. Вкладка **«Правила»** / Rules
2. Вставьте содержимое файла `storage.rules` из репозитория:
   https://github.com/Prokli81/desktop-tutorial/blob/main/storage.rules
3. **«Опубликовать»**

Правила разрешают загружать фото **только в свою папку** (`progressPhotos/ваш-id/`).

---

## Шаг 3. Проверка в приложении

1. Войдите в MyFitClub (Firebase + Firestore)
2. Вкладка **«Дневник»**
3. Заполните тренировку → выберите **фото** → **Добавить**
4. Фото появится в списке записей

---

## Лимиты (бесплатно)

- Spark даёт **5 ГБ** хранилища и **1 ГБ/день** скачивания — для фото прогресса клуба более чем достаточно
- Фото сжимаются перед загрузкой (экономия места)

---

## Если ошибка при загрузке

1. Storage включён? (шаг 1)
2. Правила опубликованы? (шаг 2)
3. Вы вошли через **email Firebase**, не демо-режим?
