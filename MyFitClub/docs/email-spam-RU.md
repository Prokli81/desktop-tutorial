# MyFitClub: письма не в спам (mail.ru и другие)

Письма о сбросе пароля отправляет **Firebase**. Чтобы они чаще попадали во
**Входящие**, сделайте настройки ниже (один раз).

---

## Шаг 1. Русский текст письма в Firebase

1. Откройте:
   https://console.firebase.google.com/project/myfitclub-4d1b8/authentication/emails
2. Выберите **«Сброс пароля»** / Password reset
3. Измените:
   - **Тема:** `Восстановление пароля MyFitClub`
   - **Текст:** можно оставить как есть или написать по-русски, например:

```
Здравствуйте!

Вы запросили сброс пароля для MyFitClub.
Нажмите ссылку ниже, чтобы задать новый пароль:

%LINK%

Если вы не запрашивали сброс — просто удалите это письмо.

С уважением,
MyFitClub
```

4. **Сохранить**

---

## Шаг 2. Имя отправителя (по желанию)

В том же разделе **Шаблоны писем** можно задать имя отправителя:
**MyFitClub** вместо технического названия.

---

## Шаг 3. На mail.ru — один раз вручную

Когда пришло первое письмо:

1. Откройте папку **Спам**
2. Найдите письмо от Firebase / noreply
3. Нажмите **«Не спам»**
4. Добавьте отправителя в **Контакты**

После этого следующие письма чаще идут во **Входящие**.

---

## Шаг 4. Проверка

1. На сайте: **Вход** → email → **«Забыли пароль?»**
2. Подождите 1–3 минуты
3. Проверьте **Входящие**, потом **Спам**
