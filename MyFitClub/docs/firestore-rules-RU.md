# MyFitClub: правила Firestore (бесплатно, безопаснее)

Сейчас база может быть в **тестовом режиме** — любой с ключом может читать/писать.
Правила ниже ограничивают доступ: **только вошедшие пользователи**.

---

## Шаг 1. Открыть правила

1. https://console.firebase.google.com/project/myfitclub-4d1b8/firestore/rules
2. Вкладка **«Правила»** / Rules

---

## Шаг 2. Вставить текст

Скопируйте **весь** файл `firestore.rules` из репозитория GitHub  
(папка проекта, корень — рядом с `MyFitClub/`).

Или откройте на GitHub:
https://github.com/Prokli81/desktop-tutorial/blob/main/firestore.rules

---

## Шаг 3. Опубликовать

1. Вставьте правила в редактор Firebase
2. Нажмите **«Опубликовать»** / Publish
3. Подождите 1 минуту

---

## Что дают правила

| Коллекция | Кто может |
|-----------|-----------|
| users | Читать все вошедшие; писать только свой профиль |
| messages, chats | Читать и писать вошедшие |
| workoutLogs | Только свои записи дневника |
| bookings | Только свои записи на занятия |

**Цена:** 0 ₽. Это не платный тариф.

---

## Если что-то перестало работать

Напишите, на каком экране ошибка. Чаще всего нужно **войти заново** в аккаунт Firebase.
