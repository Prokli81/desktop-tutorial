# MyFitClub: план развития (только бесплатно)

Развиваем клуб **без платных сервисов**. Следующие шаги по приоритету.

---

## Уже готово

- Сайт + PWA на телефон
- Вход по email, чаты в облаке
- PIN-защита, уведомления о чатах (со звуком)
- Админка, дневник, восстановление пароля

---

## Ближайшие шаги (бесплатно)

1. **Тест с 2–3 людьми** — общий чат, уведомления (`invite-friend-RU.md`)
2. **Правила Firestore** — выложить `firestore.rules` в Firebase Console (безопаснее тестового режима)
3. **Шаблон письма** — русский текст сброса пароля (меньше спама на mail.ru)
4. **Фото прогресса** — загрузка в Firebase Storage (бесплатный лимит Spark)

---

## Не делаем (платное)

- SMS / телефон
- Firebase Blaze + Cloud Functions
- Google Play / App Store

См. `BESPLATNO-RU.md`

---

## Как обновления попадают на телефон

После изменений на GitHub подождите 3–5 минут → откройте приложение с иконки → при необходимости потяните экран вниз. Чаты обновляются сразу из облака.
