# MyFitClub: что работает сейчас

Все функции ниже **бесплатны** (GitHub Pages + Firebase Spark).

---

## Работает полностью

| Раздел | Что умеет |
|--------|-----------|
| **Регистрация / Вход** | Email + пароль, коды CLIENT2026 / TRAINER2026 / ADMIN2026 |
| **Чаты** | Общий чат, личные, группы — в облаке Firestore |
| **Уведомления** | О новых сообщениях (кнопка на вкладке «Чат») |
| **Расписание** | Занятия, запись |
| **Дневник** | Записи тренировок, фото прогресса (нужен Storage) |
| **Профиль** | Роль, смена пароля, PIN-защита (по желанию) |
| **Админка** | Коды, пользователи, объявления, расписание, группы |
| **Телефон** | Установка на главный экран (PWA) |

---

## Демо-режим

**Убран с экрана входа** — поля пустые, только подсказки (placeholder).

Локальный запасной режим (без Firebase) остаётся **только** если нет интернета или облако недоступно — для разработки. В обычной работе клуба используется **только Firebase**.

---

## Опционально (не обязательно)

- FCM push при полностью закрытом приложении — платный Blaze
- SMS / телефон — не делаем
