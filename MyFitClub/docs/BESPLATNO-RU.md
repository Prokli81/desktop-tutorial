# MyFitClub: всё бесплатное — что включено

Краткий список: **что уже работает без денег** и что сознательно **не делаем**, чтобы не платить.

---

## Уже бесплатно и работает

- **Сайт и приложение** — GitHub Pages, 0 ₽
- **Регистрация и вход** — Firebase Spark (бесплатный тариф)
- **Чаты, расписание, дневник, админка** — Firestore в бесплатных лимитах
- **Восстановление пароля** — письмо на email (бесплатно)
- **Установка на телефон** — PWA, без магазина
- **PIN-защита** — в приложении, без серверов
- **Уведомления в браузере** — когда приложение открыто или в фоне

Лимиты Firebase Spark щедрые для небольшого клуба (десятки–сотни пользователей).

---

## Намеренно не используем (платное)

1. **Телефон / SMS** — каждое SMS стоит денег  
   → используем **email**

2. **Firebase Blaze + Cloud Functions для push** — платный тариф  
   → используем **уведомления браузера** (бесплатно)

3. **Google Play / App Store** — регистрация разработчика стоит денег  
   → ставим **PWA на главный экран** (бесплатно)

4. **Платные SMS-сервисы, Twilio и т.п.** — не подключаем

---

## Если лимиты Firebase закончатся

Для маленького клуба это маловероятно. Если вырастете:

- оптимизировать данные в Firestore;
- или перейти на Blaze (платно по факту использования).

Для старта **Spark достаточно**.
