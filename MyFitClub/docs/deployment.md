# MyFitClub: как открыть по ссылке в интернете

Пошаговая инструкция для этапа 1 — **GitHub Pages**.

## Шаг 0. Сделать репозиторий публичным (обязательно)

На бесплатном GitHub Pages **не работает** для приватного репозитория.

Если в **Settings → Pages** написано:

> Upgrade or make this repository public to enable Pages

сделай так:

1. Открой: https://github.com/Prokli81/desktop-tutorial/settings
2. Внизу слева открой **General** (если ты не на главной странице настроек).
3. Прокрути до **Danger Zone**.
4. Нажми **Change repository visibility**.
5. Выбери **Make public**.
6. Подтверди название репозитория.

Это бесплатно. Код станет виден всем в интернете, но без ссылки на репозиторий
люди его обычно не находят.

## Шаг 1. Включить GitHub Pages

1. Открой: https://github.com/Prokli81/desktop-tutorial/settings/pages
2. В **Build and deployment → Source** выбери **GitHub Actions** (не Deploy from branch).
3. Сохрани, если GitHub попросит.

## Шаг 2. Запустить публикацию

Код MyFitClub уже в ветке `main`.

1. Открой: https://github.com/Prokli81/desktop-tutorial/actions
2. Найди workflow **Deploy MyFitClub to GitHub Pages**.
3. Если он ещё не запускался — нажми **Run workflow** → ветка `main` → **Run workflow**.
4. Дождись зелёной галочки (успех).

## Шаг 3. Открыть сайт

После успешного деплоя открой:

```text
https://prokli81.github.io/desktop-tutorial/
```

Точную ссылку GitHub также покажет в **Settings → Pages** вверху страницы.

## Демо-вход

1. Пролистай онбординг (3 экрана) или нажми **Пропустить**.
2. Нажми **Войти по коду**.
3. Вкладка **Вход**:
   - email: `anna@myfitclub.demo`
   - пароль: `fitclub`

Или регистрация с кодом `CLIENT2026`.

## Установить как приложение на телефон

После открытия ссылки:

- **Android / Chrome:** меню → **Установить приложение** / **Добавить на главный экран**
- **iPhone / Safari:** Поделиться → **На экран «Домой»**

## Если сайт не открывается

Проверь по порядку:

1. Репозиторий **публичный**?
2. В Pages выбран **GitHub Actions**?
3. Workflow завершился **успешно** (зелёный)?
4. Прошло 1–2 минуты после деплоя?
